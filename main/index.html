Otimizador Aoxy v1

Otimizador Aoxy é um aplicativo simples e intuitivo criado em Python (Tkinter) com 11 funções que ajudam você a melhorar o desempenho do computador. Ele libera espaço, otimiza o sistema, facilita o acesso a atalhos de configurações e ainda permite criar um ponto de restauração com apenas um clique. Cada função traz uma descrição interativa ao passar o mouse, tornando o uso fácil até para iniciantes.
